====== Formats ======
^{Created Sunday 31 March 2019 @author Charles Bedon}

This subsection describes the different formats (mostly XML or JSON) used to store views, configurations or to serialize objects or to represent messages between componentes of the application.

* [[+Views]]: Describes how the graphical representations (usually nodes and connections between them) of selected objects are stored. Examples are: Object view, SDH topologies, MPLS topologies, E2E views, etc.
