====== Glossary ======
Created Thursday 04 April 2019

* Process instance: a process instance is a node stored in the data bases create using the meta data defined by a process definition
* Process definition: a process definition is a set of activity definition and process KPIs actions
* X Entity Manager: The entitymanager are used to manipulate the database, there are three: MetadataEntityManager, BusinessEntityManager, ApplicationEntityManager
